# TuneFlow Extension Icons

This directory should contain icon files for the extension:

Required sizes:
- icon-16.png (16x16 pixels)
- icon-32.png (32x32 pixels)
- icon-48.png (48x48 pixels) - Used in notifications and default
- icon-128.png (128x128 pixels) - Used in Chrome Web Store

## Creating Icons

For development, you can create simple icons using any image editor or online tool.

### Quick Online Tools:
1. https://www.favicon-generator.org/
2. https://www.canva.com/create/favicons/
3. https://realfavicongenerator.net/

### Recommended Design:
- Music-themed icon (note, headphones, waveform, etc.)
- Green color scheme (#22c55e or similar)
- Simple, clean design
- Transparent background

### Placeholder:
For development, you can use any PNG files renamed to the required sizes.
