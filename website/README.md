# TunePort Website

This directory contains the landing page for TunePort.

## Deployment

### GitHub Pages
1. Go to repository Settings -> Pages.
2. Select Source: `Deploy from a branch`.
3. Select Branch: `main`, Folder: `/website`.
4. Click Save.

The site will be live at `https://microck.github.io/tuneport/`.

## Local Development
Open `index.html` in your browser. No build step required (uses Tailwind via CDN).
